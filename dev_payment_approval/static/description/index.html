Payment/Voucher Double Approval Workflow

DevIntelle Consulting Service Pvt Ltd

Overview

When processing payment in odoo sometime it's require approval process for the solution odoo app will be set amount based approvals on payments. You can configure first payment approval amount and whom to approve it and same way second payment approve amount and whom to approve it into configuration.

Feature

Set Three Step approval verification on Payments/Voucher
Set a limit amount on Payments as Two and Three Step Verification
If Payment's Amount exceeds the pre-defined limit of First Approval,then Flow will be Two Step Verification
If Payment's Amount exceeds the pre-defined limit of Second Approval,then Flow will be Three Step Verification

How It Works

Add Users into Administration Process

Only User with following groups can Confirm Payments based on conditions

Configure Approval into Account Setting

Note: Three Step can be enabled only when Two Step(First Approval) is Enabled.

If Amount of Payment exceeds the limit of First Approval then flow will be Two Step

If Amount of Payment exceeds the limit of Second Approval then flow will be Three Step

Account User : Not Included in any Approval Group

Payment will be into First Approval, because Amount of payment is greater than pre-defined First Approval Limit

Account Manager : Included in First Approval Group

Payment will be into Second Approval, because Amount of payment is greater than pre-defined Second Approval Limit

Account Billing Manager : Included in Second Approval Group

Payment will be Posted Now
